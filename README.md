# 🧠 Python CLI Quiz Game

A command-line quiz application built using Python.

## 🚀 Features

- Multiple choice questions
- Randomized question order
- 15-second timer per question
- Input validation
- Score tracking
- Percentage calculation
- Persistent high score using file handling

## 🛠 Technologies Used

- Python
- Random module
- Time module
- File handling
